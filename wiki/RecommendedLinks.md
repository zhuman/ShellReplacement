# Recommended Links



Here is a list of useful websites for building shell replacements:



* [Litestep Developer Site and CVS](http://lsdev.org/news.php)

* [Source code for the ReactOS Explorer replacement](http://shell.franken.de/~sky/explorer-doc/main.html)

* [A listing of all functions documented by Microsoft after a legal settlement](http://msdn2.microsoft.com/en-us/library/ms807073.aspx)

* [Xoblite - An open-source shell replacement](http://xoblite.net/)

* [A study of the Windows shell APIs covering hundreds of undocumented functions](http://www.geoffchappell.com/viewer.htm?doc=studies/windows/shell/)

* [CSIDLs - An MSDN article about system folders](http://msdn2.microsoft.com/en-us/library/ms649274.aspx)

* [An MSDN article about per-user shells](http://msdn2.microsoft.com/en-us/library/ms838576.aspx)

* [Shell-Shocked - A listing of many different shell replacements](http://shell-shocked.org)

* [A Shell-Related Mailing List](http://www.freelists.org/list/shell-coding)

* [Wez's Evil Shell Replacement](http://evildesk.netevil.org/index.php)
